## Summary
Search does not return relevant results when a query contains a typo.

---

## Application
- Website: https://avto.pro/
- Component: Main Search

---

## Environment
- OS: macOS Tahoe 26.1
- Browser: Safari 26.1

---

## Steps to Reproduce
1. Open the website
2. Enter a spare part name with a typo (e.g. "termostat" instead of "thermostat")
3. Observe the search results

---

## Expected Result
Search recognizes the typo and displays relevant results for the correct spare part.

---

## Actual Result
Search displays unrelated spare parts.

---
