# User Stories – Search Functionality

## US-01
As a user, I want to enter a spare part name into the search field so that I can quickly find the required part.

## US-02
As a user, I want to see autocomplete suggestions while typing in the search field so that I can select an existing spare part faster.

## US-03
As a user, I want to select a spare part category in the filter so that search results are limited to the selected category.

## US-04
As a user, I want to search by car brand so that I see spare parts related only to that brand.

## US-05
As a user, I want to search by VIN code so that I get a list of spare parts compatible with my vehicle.

## US-06
As a user, I want to search by spare part serial number (article code) so that I can find a specific part directly.

## US-07
As a user, I want to search using a product code so that I can quickly find the required item.

## US-08
As a user, I want the search to handle spelling mistakes and still show relevant results.

## US-09
As a user, I want to clear the search field with one click so that I can enter a new query easily.

## US-10
As a user, I want to see a “No results found” message so that I understand the query is incorrect or the product is unavailable.

## US-11
As a user, I want to apply filters (price, brand) so that search results are more relevant.

## US-12
As a user, I want to see the number of search results so that I can better evaluate the available options.
